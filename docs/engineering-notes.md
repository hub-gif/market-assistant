## 任务执行模型

- **当前实现**：创建京东流水线任务后，后端在 **后台线程** 中执行采集与报告步骤。  
- **含义**：适合**本机或单进程演示**；进程重启会中断未完成任务。  
- **生产演进建议**：改为 **Celery / RQ / Django-Q** 等任务队列 + 独立 worker 进程；并配置 **超时、重试、并发上限** 与日志采集。
